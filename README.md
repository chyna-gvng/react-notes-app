### React Notes App
Markdown notes app:
- React
- Vite

### Dev Setup
- Clone repo:  
    ```
    git clone https://github.com/chyna-gvng/react-notes-app.git
    ```
- Change directory:  
    ```
    cd react-notes-app
    ```
- Install modules & requiremnts:  
    ```
    npm install
    ```
- Start app:  
    ```
    npm start
    ```
